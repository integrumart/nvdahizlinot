\# Quick Note Taker (Hızlı Not Al)

\*\*Developer:\*\* Volkan Özdemir Yazılım Hizmetleri



A simple, fast, and lightweight note-taking add-on for NVDA users. This tool allows you to quickly save notes with timestamps directly to your desktop without leaving your current workflow.



\## Features

\* \*\*Global Shortcut:\*\* Press `NVDA+Shift+N` from anywhere.

\* \*\*Smart Saving:\*\* Automatically creates or appends to a file named `Notlarim.txt` on your Desktop.

\* \*\*Timestamps:\*\* Each note is saved with the current date and time for easy tracking.

\* \*\*Multi-line Support:\*\* Write long notes using a standard multi-line edit box.

\* \*\*Developer Support:\*\* Includes a donation link (PayTR) to support ongoing development for visually impaired users.



\## How to Use

1\. Press `NVDA+Shift+N`.

2\. Type your note in the dialog box.

3\. Use the \*\*"Tamam" (OK)\*\* button to save or \*\*"İptal" (Cancel)\*\* to discard.

4\. You can also use the \*\*"Bağış Yap" (Donate)\*\* button to support the developer via a secure PayTR link.
